# ALLUX Bills Scanner

A Python tool to automatically parse **CFE electricity bills** and extract structured data for energy analytics in commercial centers.

This project is part of the **ALLUX Tenant Energy Characterization initiative**, which aims to build digital energy profiles for malls and large commercial facilities.

---

## Features

- Batch processing of CFE bill PDFs
- Automatic extraction of:
  - Tariff
  - Consumption (kWh)
  - Demand (kW)
  - Billing period
  - Meter information
- Generation of structured CSV datasets
- Designed for large-scale analysis (hundreds or thousands of bills)
